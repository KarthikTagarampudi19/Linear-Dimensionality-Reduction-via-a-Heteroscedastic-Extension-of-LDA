Instructions to execute the project file.


1)MATLAB code files will be under "MatlabCode" folder -> techniques.
2)PyTorch code files will be under "PyTorchCode" folder.




To run MATLAB code:

We need to run Glass Data and Landsat Data in MATLAB

In the Command prompt 

Command to run Glass Data -  hlda_GlassData_run

Command to run Landsat Data - hlda_LandSatData_run



To run PyTorch code:


We need to run Glass Data and Landsat Data in PyTorch

In the Command prompt 

Command to run Glass Data - python3 hlda_GlassData_run.py

Command to run Landsat Data - python3 hlda_LandSatData_run.py
